# オンラインのオセロゲーム

ブラウザ上で動くオンラインオセロゲームです。  
1対1のリアルタイム通信で対戦が可能です。

- スマホ・PC 両対応  
- サーバーとクライアントを分離設計  
- WebSocket による双方向通信  

## 技術構成

- クライアント: HTML / CSS / JavaScript (Vanilla)  
- サーバー: FastAPI + WebSocket  
- 状態管理: Redis（盤面、ユーザーIDの一時保存による「再接続」処理）  
- デプロイ: GitHub Pages（クライアント）/ Render（サーバー）/ Upstash（Redis）
